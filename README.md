## Usage

#### Manual Installation

Clone repo, install pods and open `Antidote.xcworkspace` file with Xcode 12+.

```
git clone https://github.com/Antidote-for-Tox/Antidote.git
cd Antidote
pod install
open Antidote.xcworkspace
```
